ScopedLabs changelog

A running record of major ScopedLabs updates, feature improvements, and platform polish.

April 2026

Knowledge Base Engine V1 completed

Added a reusable Knowledge Base system across all active tool categories. Each tool now has a guide panel with practical explanations, input descriptions, common mistakes, planning assumptions, and professional responsibility notes.

  • Added Knowledge Base guides across all 10 active categories.
  • Added expandable input descriptions for tool-specific terms and assumptions.
  • Standardized the collapsed Knowledge Base card layout across tool pages.
  • Kept help content clean and page-friendly without inline question-mark clutter.
April 2026

Export reports and saved snapshots added

Added report-style documentation features so calculations can be preserved, printed, exported, or saved for later review. Pro users can document inputs, assumptions, outputs, guidance, and chart snapshots where supported.

  • Added exportable planning reports across active tool categories.
  • Added saved report snapshots tied to signed-in accounts.
  • Added Account page saved report viewer with view and delete actions.
  • Connected snapshot saving through the existing Supabase and Cloudflare Worker path.
April 2026

Guided Design Flow messaging improved

Updated the site to better explain that ScopedLabs is more than a collection of isolated calculators. Guided Design Flows connect related tools so users can start with a baseline, carry useful values forward, and document planning decisions.

  • Added Guided Design Flow explanation to the home page.
  • Added Guided Design Flow explanation to the About page.
  • Normalized category page pipeline wording and CTAs.
  • Improved Tools page copy to mention focused calculators, guided flows, and reports.
April 2026

Navigation and trust pages polished

Refined the main navigation, added sticky header behavior, and added a planning-aid disclaimer page so site expectations are clearer for visitors.

  • Added browser-style navigation tabs.
  • Added active tab highlighting for the current page area.
  • Made the header navigation sticky while scrolling.
  • Added the Planning-Aid Disclaimer page.
  • Added footer links to important utility pages.

What may improve next

ScopedLabs will continue improving tool clarity, documentation quality, and formula confidence. Future updates may include deeper Knowledge Base content, formula verification passes, launch-readiness cleanup, SEO/indexing verification, and expanded reporting features.
