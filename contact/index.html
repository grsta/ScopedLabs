Helpful context

What to include

If you are reporting a calculator issue, include the tool name, the inputs used, the output you received, and what looked wrong or unclear.

  • Tool category and tool name.
  • Inputs or assumptions used.
  • Screenshot or copied result text if available.
  • Whether the issue is visual, mathematical, export-related, or account-related.
