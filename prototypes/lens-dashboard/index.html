Engineering Interpretation

At 80 ft with a target width of 20 ft and a 6.40 mm sensor, the raw geometry calls for a long-range focal length. This narrows field of view and reduces layout flexibility.

Corrective Guidance / Path to Healthy

Why this status

The current combination of distance, target width, sensor width, and upstream detail requirement is increasing lens pressure.

Likely drivers

  • Long-range focal demand is driven by distance-to-target and target-width relationship.
  • The design is asking one camera/lens combination to hold a tight field of view.

Possible planning actions

  • Move the camera closer to the target area.
  • Increase acceptable target width if the use case allows it.
  • Split the view across more than one camera.

Healthy Target

Bring lens-selection pressure back into the preferred planning band while preserving required scene detail.

Follow-up Checks

  • Validate focal length against manufacturer FOV chart.
  • Confirm mounting location and alignment tolerance.
  • Verify sensor format compatibility.

Revision Triggers

  • Camera mounting location changes.
  • Target width or scene objective changes.
  • Sensor or lens family changes.
