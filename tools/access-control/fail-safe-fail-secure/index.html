Fail-Safe vs Fail-Secure

This tool starts the Access Control design flow

Use this first to determine how a door should behave during power loss before you choose reader strategy, lock power, panel sizing, and long-term access structure.

This is the logic gate for the rest of the access-control lane, because the wrong fail state can create life-safety conflicts or security exposure before hardware selection is even finalized.
