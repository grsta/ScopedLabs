Scope first, then calculate.

Different doors and zones can have different egress, fire release, security, reader, lock, power, and policy requirements. The planner stores the active scope so downstream tools can write guidance to the correct door or zone.

Core flow
  1. Define access scope.
  2. Classify egress and fire behavior.
  3. Run Fail-Safe / Fail-Secure.
  4. Continue through reader, power, panel, and summary.
