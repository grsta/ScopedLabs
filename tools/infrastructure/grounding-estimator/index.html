Grounding Estimator

Analyzer Tool

Size grounding intent before the site turns it into a field problem

This tool gives planning guidance for grounding conductor scale and grounding-target strictness so you can quickly tell whether the job still looks routine or is drifting into a more engineered grounding strategy.

It is a standalone infrastructure analyzer, not part of the current design lane.

Best for: quickly sizing a grounding conductor and checking whether the grounding target is conservative, normal, or likely to require enhanced electrode strategy.
