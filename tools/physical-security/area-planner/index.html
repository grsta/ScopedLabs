Area / Zone Planner

Part of a Design Flow

Define the areas before running the Physical Security flow

Real sites rarely use one distance, one lens, and one camera-count assumption everywhere. Use this step to define one or more areas so downstream coverage, spacing, detail, and lens results can be tied to the correct zone.

Best for: splitting a property into planning areas so each zone can carry its own distance, HFOV, protected length, camera count, spacing, detail target, and lens validation path.
