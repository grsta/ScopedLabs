This tool continues the Physical Security design flow

Use this step to verify that the spacing plan still produces continuous real-world coverage once overlap is applied and effective footprint is reduced to something you can actually trust.

This is where a layout that looks evenly spaced on paper gets checked for actual uncovered width before you move into subject-detail validation.

Best for: validating that your spacing and overlap plan truly removes blind spots before moving into pixel-density verification.
