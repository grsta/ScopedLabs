# Battery Bank Sizer

**Category:** power  
**Path:** /tools/power/battery-bank-sizer/  
**Access:** Free  

## What it does
Sizes a DC battery bank using:
- Load (W)
- Runtime (hours)
- Voltage (V)
- Depth of discharge (%)
- Efficiency (%)

Outputs:
- Required Wh
- Required Ah @ Voltage
