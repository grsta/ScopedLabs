Failure Runtime Loss

Best for: quantifying how much backup runtime disappears when the system enters a degraded or higher-load failure state.

What this models

  • Normal runtime versus degraded-state runtime from the same installed battery reserve
  • How much additional failed-state load cuts usable survivability
  • Whether backup resilience remains stable, moderate, or severely degraded during failure

Model: runtime is based on usable battery energy after efficiency and max depth-of-discharge are applied, then compared between baseline and failed-state load conditions.

Notes / Assumptions
  • Runtime = (Battery Wh × Efficiency × Usable DoD) ÷ Load
  • Failure scenario assumes load rises under degraded conditions
  • This tool compares survivability under failure, not charger recovery time
