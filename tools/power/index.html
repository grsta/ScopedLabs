Power & Runtime Planning Guide

A practical overview of load conversion, runtime planning, battery sizing, growth margin, and branch assumptions before selecting backup power equipment.

Read the guide

Power problems usually show up as undersized runtime, overloaded circuits, surprise growth, or “it should hold” assumptions that collapse under a real outage. These tools help you convert electrical load into practical runtime, capacity, and resilience planning before the design gets boxed in by bad assumptions.

Guided Design Flow

Start with the first step. Build toward the full design.

Some ScopedLabs tools work as a connected workflow. In Power & Runtime, the Design Flow moves from baseline load conversion into projected growth, runtime impact, and final storage sizing so the power design is checked as one practical system instead of disconnected estimates.

GUIDED DESIGN FLOW
VA / Watts / Amps → Load Growth → UPS Runtime → Battery Bank Sizer

The flow includes both Free and Pro tools. Pro access unlocks the full sequence for deeper planning, exportable reports, and saved design snapshots.
